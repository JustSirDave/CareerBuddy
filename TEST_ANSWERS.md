## 🎯 QUICK TIPS FOR TESTING

1. **For Resume/CV:** Copy each answer one at a time and paste into the bot
2. **For Multiple Items:** After each experience/education/project, wait for bot confirmation before sending "done"
3. **For Bullets:** Send one bullet point at a time, then type "done" when finished
4. **For Skills:** Wait for AI suggestions, then select by numbers (e.g., "1,3,5,7,9")
5. **For Final Generation:** Type "yes" when previewing, and choose format (DOCX or PDF if premium)

## 📋 DOCUMENT TYPE CHOICES

When starting, the bot may ask what type of document you want:

- **Resume:** Type `resume` or `choose_resume`
- **CV:** Type `cv` or `choose_cv`
- **Cover Letter:** Type `cover` or `choose_cover`
- **Revamp:** Type `revamp` (requires uploading existing document)
